# Tutorial

This tutorial will demonstrate how to build a simple user registration form with Fiora.

#### 1. Create a form

First thing we will need to do import the factory function and create an instance of fiora form.


```js-static
import createFiora from 'fiora';

const { Form, Field } = createFiora();
```

`Form` and `Field` are the two essential components we need to start building.
